What is Dart?

A new programming language for the Web

Class based (not prototypal), object oriented, optionally typed

Uses a source code VM

Using Dart

  • Compiled as JavaScript with dart2js
  • Through the Dart VM (currently only Dartium)
  • Standalone through the command line

What does Dart consist of?

  • Dart VM: Run Dart code natively (only present in Dartium currently).
  • dart2js: Compiles Dart code to JavaScript for use in browsers without the VM
  • Dart Editor: A full IDE built just for Dart based on Eclipse

Why Dart?

The goal is to make it easier to develop larger, faster, and more complex Web apps

  • Better performance: The Dart VM is faster than handwritten JS!
  • Better productivity: better standard libraries, easier to learn (fewer pitfalls)

“Obsoleted” third-party JavaScript libraries

“Batteries included”

  • Moment.js (DateTime, Stopwatch)
  • jQuery (querySelector(), querySelectorAll() - read more)
  • Q (Future<T>)
  • Node.js (dart:io)
  • Any AJAX library (HttpRequest)
  • Jasmine, QUnit, etc. (package:unittest/unittest.dart)
  • i18n libraries (intl)

Code example: HTTP


import 'dart:html';
import 'dart:convert';

void main() {
  var url = 'weather.json';
  HttpRequest.getString(url).then(displayWeather, onError: (error) => print(error));
}

void displayWeather(result) {
  print(result);
  Map weather = JSON.decode(result);
  var temp = weather['currently']['temperature'];
  var feelsLike = weather['currently']['apparentTemperature'];
  var condition = weather['currently']['summary'];
  var buffer = 'The current temperature in your area is $temp&deg;, '
  buffer += 'but it feels like $feelsLike&deg;
';
  buffer += 'The current condition in your area is $condition';
  querySelector('#temperature').innerHtml = buffer;
}

Code example: Method cascading


  import 'dart:html';

  void main() {
  var address = getAddress()
    ..setStreet(“Elm”, “13a”)
    ..city = “Carthage”
    ..state = “Eurasia”
    ..zip(66666, extended: 6666);
  // this is equivalent to
  var address = getAddress();
  address.setStreet(“Elm”, “13a”);
  address.city = “Carthage”;
  address.state = “Eurasia”
  address.zip(66666, extended: 6666);
}

Code example: Classes


import 'dart:math';

class Point {
  num x;
  num y;
  Point(this.x, this.y);

  num distanceTo(Point other) {
    var dx = x - other.x;
    var dy = y - other.y;
    return sqrt(dx * dx + dy * dy);
  }
}

Is Dart ready for prime time?

  • I could only find one company other than Google using it in production
  • The AngularDart “Hello {{yourName}}” demo crashed my browser, so don't touch that thing
  • Nobody can run your Dart code natively
  • dart2js files can be huge

The bottom line

Until the Dart VM is integrated into Chrome Stable, you probably shouldn't use it.
